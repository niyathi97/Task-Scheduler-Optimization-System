# Task Scheduler Optimization System

A Python-based task scheduling system that optimizes task execution using Priority Queue and Greedy Scheduling.

## Features

- Task Scheduling
- Priority Queue (Heap)
- Greedy Algorithm
- Profit Calculation
- Timeline Generation
- Dashboard with Charts

## Technologies

- Python
- Tkinter
- Matplotlib

## Run

python gui.py

## DSA Concepts

- Heap
- Priority Queue
- Greedy Algorithm
- Sorting
